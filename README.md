## 🚀 How to Start

1. 이 레포지토리를 클론합니다.

   ```bash
   git clone https://github.com/2024-Winter-Bootcamp-team-K/project-repo.git
   ```

2. Docker를 사용하여 프로젝트를 실행합니다.

   ```bash
   docker-compose up --build
   ```

3. 브라우저에서 [http://localhost:3000](http://localhost:3000)를 열어 확인합니다.
